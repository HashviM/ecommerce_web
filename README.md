# Cute Fashion – Custom Tailoring E-Commerce Website

A premium frontend fashion boutique website designed for custom women tailoring services.  
This project focuses on elegant UI design, service-based workflow, and fashion catalog experience.

## ✨ Features

- Custom tailoring design catalog (Salwar, Kurti, Maxi, Ready-to-Wear)
- Fabric showcase section
- Tailoring service workflow visualization
- Pricing estimation UI model
- Modern fashion boutique layout
- Fully responsive design
- Clean premium aesthetic

- Home

## 🎨 UI / UX Focus
